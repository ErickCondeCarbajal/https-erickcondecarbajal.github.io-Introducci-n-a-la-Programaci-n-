Cuarto Ejercicio

4. Escribir un Programa que calcule 4 áreas: 1-cuadrado, 2-rectángulo, 3-triángulo y 4-círculo. El programa deberá solicitar para cada caso el ingreso de los datos necesarios para la definición de cada área, en caso se ingrese una opción diferente envié el mensaje de que la operación no está permitida. (considere el valor de la función de Math.PI para el cálculo del área del circulo Math.PI=3.141592653589793).
